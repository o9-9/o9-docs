* Install the most used **Windows Features** by checking the checkbox and clicking "Install Features" to install them.

* All .Net Framework
* HyperV
* Legacy Media
* NFS Network 
* Enable Search Web
* Disable Search Web
* Enable Registry Backup
* Enable F8 Boot
* Disable F8 Boot
* Linux
* Enable Sandbox

### Fixes
* Quick fixes for your system if you are having issues.

* Enable Autologin
* Reset Update
* Reset Network
* System Scan
* WinGet Reinstall
* Remove Adobe

### Legacy Windows Panels

Open old-school Windows panels directly from o9. Following Panels are available:

* Control
* Network
* Power
* Region
* Sound
* System
* User Accounts

### Remote Access

Enables OpenSSH server on your windows machine.
